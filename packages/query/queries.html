Queries

These are all Query hooks.

Checkout the <FhirQueryLoader /> component.
It is meant to work in tandem with query hooks, and handles the loading / error / display lifecycle automatically.

📄️ useFhirRead

Read a FHIR resource

📄️ useFhirVRead

Read a version of a FHIR resource

📄️ useFhirSearch

Search for FHIR resources

📄️ useFhirSearchOne

Search for a single FHIR resource

📄️ useFhirSearchAllPages

Get all the search results at once

📄️ useFhirInfiniteSearch

Search for FHIR resources

📄️ useFhirExecute

Execute an operation that does not affect state

📄️ useFhirGraphQL

Execute a Graphql query

📄️ useFhirGraphQLResult

Execute a Graphql query and get raw Graphql Response

📄️ useFhirHistory

Retrieve previous versions of a FHIR resource

📄️ useFhirClient

Execute a series of FHIR operations

📄️ useFhirCapabilities

Read FHIR server capabilities
